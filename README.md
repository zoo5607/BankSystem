## 구현 기능

### 기능 1  
이름, 계좌, 비밀번호를 입력받아 계좌를 생성한다.

### 기능 2  
자신의 계좌를 조회할 수 있고 다른 계좌로 입금, 출금이 가능하다.

### 기능 3  
계좌의 입금, 출금내역을 확인할 수 있고, 날짜가 자동으로 기입된다.

### 기능 4
정규표현식에 따라 계좌를 작성하고  뒷자리는 마스킹 기능을 추가하여 *로 표시한다.
